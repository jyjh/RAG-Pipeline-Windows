New User Guide

Use this app to turn trusted local PDFs into a searchable engineering knowledge base, review extracted content, and ask source-cited questions from the local index.

Core Workflow

  1. Add PDFs Upload team reports, textbooks, notes, papers, or standards and choose a reliability group for each PDF.
  2. Wait for jobs Ingestion extracts Markdown and assets; indexing builds the retrieval database in the background.
  3. Review sources Tag highlighted ungrouped PDFs, check quality badges, trust status, downloads, and extracted index rows before relying on answers.
  4. Ask questions Use the chat panel for evidence-backed answers, then inspect Sources and Tool results.

Intended Use

  • Keep FSAE knowledge transfer local and reproducible across uploaded source documents.
  • Prefer reviewed, approved sources for design decisions, calculations, and team documentation.
  • Use web search only as supporting context when local PDFs are incomplete or outdated.

Review Checklist

  • Confirm each PDF has useful extracted text and nonzero chunks.
  • Tag ungrouped PDFs as Official, Student Research, or Unofficial so retrieval ranking can weight sources correctly.
  • Open cited pages or downloads when an answer affects engineering decisions.
  • Flag stale or rejected sources so future users see the risk before relying on them.
  • Re-run a single PDF when OCR, figures, formulas, or source metadata look wrong.

Operating Notes

  • Long ingestion and indexing jobs run in the background; chat can still use the last published index.
  • Index edits are disabled while indexing is active to avoid writing into a changing database.
  • Saved chats live in this browser only and are useful for recurring investigation threads.
